# ReclAI

ReclAI es un sistema inteligente de gestión de reclamos desarrollado en Python utilizando modelos de lenguaje locales mediante Ollama.

## Características

- Registro e inicio de sesión.
- Roles de administrador y cliente.
- Configuración de empresas mediante IA.
- Registro inteligente de reclamos.
- Resumen automático de conversaciones.
- Almacenamiento en JSON.
- Arquitectura modular.

## Tecnologías

- Python
- Ollama
- Llama 3
- JSON
